# Changelog

## 0.2.0

Expanded classical cipher support.

### Added

- ROT / ROT13 solver
- ROT47 solver
- Keyword Substitution solver
- Playfair cipher solver
- Autokey Vigenere solver
- Polybius Square solver
- Baconian cipher solver
- New sample ciphertext files
- Additional unit tests

## 0.1.0

Initial working release of freqcrack.

### Added

- Frequency analysis command
- Cipher detection command
- Caesar solver
- Atbash solver
- Affine solver
- Vigenere solver
- Beaufort solver
- Rail Fence solver
- Columnar Transposition solver
- Monoalphabetic substitution solver
- Advanced substitution hill-climbing mode
- Sample ciphertext files
- Unit tests
- CLI version option
- README documentation
