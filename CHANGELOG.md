Initial working release of freqcrack.

### Added

- Frequency analysis command
- Cipher detection command
- Caesar solver
- Atbash solver
- Affine solver
- Vigenere solver
- Beaufort solver
- Rail Fence solver
- Columnar Transposition solver
- Monoalphabetic substitution solver
- Advanced substitution hill-climbing mode
- Sample ciphertext files
- Unit tests
- CLI version option
- README documentation
